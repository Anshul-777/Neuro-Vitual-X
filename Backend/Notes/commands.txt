# Neuro-Vitals System: Installation and Run Commands

This file provides a consolidated list of all necessary installation commands (individual `pip install` commands where applicable, to avoid dependency conflicts) and commands to run the Python files in the correct order of working, with exact paths.

--- General Setup: Python & VS Code ---
1.  **Install Python 3.8+**: Download from [python.org/downloads](https://www.python.org/downloads/). Add to PATH during installation.
2.  **Install VS Code**: Download from [code.visualstudio.com](https://code.visualstudio.com/).
3.  **Open VS Code Terminal**: Navigate to your desired project location (e.g., `cd path/to/Neuro-Vitals`).
4.  **Create Project Directory**: (If starting from scratch)
    ```bash
    mkdir Neuro-VX
    cd Neuro-VX
    mkdir -p Backend/Notes
    ```

--- Virtual Environment Setup ---
5.  **Create Virtual Environment**:
    ```bash
    python -m venv venv
    ```
6.  **Activate Virtual Environment**:
    *   **Windows**:
        ```bash
        .\venv\Scripts\activate
        ```
    *   **macOS/Linux**:
        ```bash
        source venv/bin/activate
        ```
    (Ensure `(venv)` appears in your terminal prompt after activation)

--- Core Library Installations (Individual `pip install` commands) ---
7.  **Install OpenCV Python**:
    ```bash
    pip install opencv-python
    ```
8.  **Install MediaPipe**:
    ```bash
    pip install mediapipe
    ```
9.  **Install NumPy**:
    ```bash
    pip install numpy
    ```
10. **Install SciPy**:
    ```bash
    pip install scipy
    ```
11. **Install Matplotlib**:
    ```bash
    pip install matplotlib
    ```
12. **Install Librosa** (for voice analysis):
    ```bash
    pip install librosa
    ```
13. **Install SoundFile** (dependency for Librosa audio operations):
    ```bash
    pip install soundfile
    ```
    *(Note: The `requirements.txt` file is located at the project root `Neuro-VX/requirements.txt` and can be used for bulk installation: `pip install -r requirements.txt`)*

--- MediaPipe Model Download & Placement ---
14. **Download Face Landmarker Model**: Save this as `face_landmarker.task` in the `Neuro-VX/` directory first.
    (You would typically copy a pre-downloaded model or use a Python script to fetch it.)
    ```python
    # Python script to download model (example, save as download_model.py in Neuro-VX/)
    import os
    import urllib.request

    model_url = 'https://storage.googleapis.com/mediapipe-models/face_landmarker/face_landmarker/float16/1/face_landmarker.task'
    model_path = 'face_landmarker.task'

    if not os.path.exists(model_path):
        print(f"Downloading {model_path}...")
        urllib.request.urlretrieve(model_url, model_path)
        print("Download complete.")
    else:
        print(f"{model_path} already exists.")
    # Then run: python download_model.py
    ```
15. **Move Model to Backend Directory**:
    ```bash
    mv face_landmarker.task Neuro-VX/Backend/
    ```
    (Alternatively, directly download to `Neuro-VX/Backend/` if possible)
